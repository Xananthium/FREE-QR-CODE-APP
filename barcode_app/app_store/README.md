## Quick Start Guide

### Before Submitting to App Store

1. **Set up website pages:**
   - [ ] Publish privacy policy at `/privacy`
   - [ ] Publish support page at `/qr-support`
   - [ ] Test both URLs are accessible

2. **Prepare visual assets:**
   - [ ] Design app icon (1024x1024)
   - [ ] Create 5 screenshots with text overlays (use 04_screenshot_overlays.md)
   - [ ] Optional: Create app preview video

3. **Complete App Store Connect setup:**
   - [ ] Create app listing
   - [ ] Copy description from `01_app_store_description.md`
   - [ ] Copy promotional text from `03_promotional_text.md`
   - [ ] Enter keywords from `02_keywords.md`
   - [ ] Set URLs: privacy and support

4. **Upload build:**
   - [ ] Archive in Xcode
   - [ ] Upload to App Store Connect
   - [ ] Wait for processing (5-10 mins)

5. **Configure privacy:**
   - [ ] Answer: "No data collected"
   - [ ] Verify "No Data Collected" badge shows

6. **Submit for review:**
   - [ ] Follow checklist in `07_submission_checklist.md`
   - [ ] Submit
   - [ ] Wait 1-3 days for review
